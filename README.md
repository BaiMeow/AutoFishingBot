# AutoFishingBot
an auto fishing robot for Minecraft server  
一个可以在 Minecraft 服务器中自动钓鱼的机器人，1.15.2

-------------

配置详见 `config.toml`  
为支持正版，仅支持正版登录

-------------

基于 [go-mc](https://github.com/MscBaiMeow/go-mc)
